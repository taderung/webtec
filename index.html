About

Diese Webseite ist im Rahmen der Testataufgabe im Modul WEBTEC an der Hochschule Luzern (Informatik) entstanden. Diese Webseite wird nach Überprüfung des Dozenten abgeschaltet. Der Source Code ist auf Github einsehbar. Die Funktionalität gewisser Elemente wurden absichtlich nicht implementiert, da sie nur zu Demonstrationszecken dienen.

Alle Bilder auf dieser Seite sind selbst erstellt worden. Diese sind per Gesetz urheberrechtlich geschüzt. Bitte die Fotos keinesfalls ohne meine ausdrückliche Erlaubnis weiterverbreiten oder herunterladen.


Vielen Dank dass sie so meine persönlichen Daten schützen.
